 The first thing I want to stress is that this does not make the game fully accessible. As of right now, you will need a cited friend in order to play through the game, though left for dead works best as a co-op shooter anyway. rest assured I am working on it but it might take awhile before the game is fully playable. 

also, you must host the lobby when playing with friends as the mod requires that to work. 

features. 

this mod features.   

•naration of the main menu.

•naration of the pause menu.

 •naration of in game events (for example coach grabbed by Hunter, Rochelle freed from charger). 

•naration of Player status, Health, equipped weapon, available first aid/throwables/pain pills, et cetera. bound to the H key by default.

•  announcement when looking at a pick up, although this is a little spotty. 

•Confirmation when picking up an item, so I would sort of advise just pressing E in the location of loot if the announcement is acting up. 

•lock-on aim which will automatically target hostile enemies. It operates on a power hierarchy system so boss infected will be targeted first, then specials, then regular zombies. This changes if the enemy is within 300 units of You at which point the lock on Will favour  them. this feature is bound to the X key, and it should be noted it does not work in the PVP game modes so as to prevent unfairness and bans  

known issues.

 some menus are impossible to make accessible. these include

  •the lobby menu (campaign, chapter, difficulty, character) although the contents of those menus are fully narrated with an announcement upon entering them of the title of the menu so this is not so much of an issue. 

•the options menus with adjustable options such as the audio menu. no real work around there unfortunately.

•addons menu, although this is easily accessible through Steam. 

•achievements menu, which is also accessible through Steam. 

Single player navigation is unfortunately not possible at this time. it is my plan to add a form of navigation assist via an audio beacon that attaches to objects listed in a scanner. I have No idea whether that would be possible however, so I figured something released was better than potentially nothing at all. 

essentially, this is a beta test while I try and get the rest of it running.
